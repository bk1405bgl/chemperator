# Chemperator

Chemperator is a web application based on laravel framework with expressive, elegant syntax. It is developed for companies in chemistry branch to manage their products, customers, orders and invoices. There will be more options to connect to online-shop and even to marketplaces like ebay and amazon.

## Who is the author?

Me, Bilal Kuzey. Worked in  the chemistry industries for about six years as an online-marketing-manager. Used Excel Sheet Files to import/export data between system and marketplaces.

## Tech Stack

Laravel (PHP) and React.js (Javascript) combined with Bootstrap (CSS) and MariaDB Database (SQL).
